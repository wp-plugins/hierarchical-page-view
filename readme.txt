== Description ==

The plugin displays the Hierarchial Page View only in the Admin section for large number of Post/Pages. Currently goes 5 Levels Deep.

== Installation ==

1. Copy the folder 'Hierarchical-Page-View' into the directory 'wp-content/plugins/'.

2. Activate the plugin.

3. Go the Pages section within the WP-Admin.

4. Click on Hierarchical Page View
